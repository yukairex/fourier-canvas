# Fourier Canvas

An interactive Fourier transform drawing canvas. Draw a closed path — the app decomposes it with DFT and animates rotating epicycles (circles-on-circles) that trace your shape.

## Features

- **Draw mode** — draw any closed curve; see it reconstructed by epicycles
- **Studio mode** — manually set frequency / amplitude / phase for each vector and watch the pattern emerge
- 6 built-in presets (heart, star, infinity, rose, square, spiral)
- N (harmonic count) and speed sliders
- Toggle: circles, vectors, original path
- Trail persistence: short / long / infinite
- 3 themes: Paper / Graphite / Oscilloscope
- 5 languages: English, 中文, 日本語, Français, Español

## Usage

Open `index.html` in any browser — no build step, no server needed.

Or visit the live page at: https://yukairex.github.io/fourier-canvas/
